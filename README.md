**Employee Management System - StaffSync**
NIBM EAD Coursework  **Index: GADSE243F-007 & GADSE243F-013**

**Project Overview**
An Enterprise Java application built using the MVC (Model-View-Controller) architecture to manage employee records, user authentication, and reporting.

**Tech Stack***
    Language: Java (JDK 17+)
    Architecture: MVC
    Database: MySQL
    IDE: IntelliJ IDEA 

  **Setup Instructions**

1.  Database Setup: 
    *   Open XAMPP/MySQL.
    *   Create a database named `Employee_Management_System`.
    *   Import the `database.sql` file included in this repository.
2.   Configuration: 
     Navigate to `src/resources/config.properties`.
    *   If your MySQL has a password, enter it in the `db.password=` field.
3.  Run:
    *   Open the project in your IDE.
    *   Add the `mysql-connector-j.jar` to your project libraries.
    *   Run `Main.java`.

 **Key Features**
  User Authentication: Secure login for system access.
  Employee CRUD: Create, Read, Update, and Delete employee data.
  MVC Pattern: Clean separation of concerns for better maintainability.
  Custom UI: Professional interface with integrated branding and backgrounds.
